Справочная информация

Работа с API Пкаско происходит по протоколу HTTP и представляет обычные REST запросы.

Для общения с API нужно иметь токен авторизации, который получается путем логина и пароля на адрес http://pkasko.ru/proservice/auth (логин и пароль приобретается через КАСКО калькулятор pkasko.ru). Токен действует две недели с момента получения.

Для базового расчета достаточно отправить запрос:
- список марок - GET http://pkasko.ru/proservice/getMakes
- список моделей - GET http://pkasko.ru/proservice/getModels
- расчет КАСКО - POST http://pkasko.ru/proservice/calculateKasko (параметры)
- расчет ОСАГО - POST http://pkasko.ru/proservice/calculateOsago (параметры)

Базовые параметры расчета:
-- make - название марки текстом, например Audi
-- model - название модели текстом, например A4
-- year - год выпуска ТС
-- power - мощность
-- price - цена
-- drivers - массив из данных с водителями, где каждый элемент массива:
--- age - возраст
--- experience - стаж
--- sex - пол (m - мужчина, w - женщина)
